Dashboard Page (HTML & CSS)
 Project Title

Dashboard Design using HTML & CSS

 Project Description

This project is a dashboard interface created using HTML and CSS only.
It displays an overview of information using cards, icons, tables, and structured layout, similar to an admin dashboard.

The focus of this task is on alignment, spacing, colors, and professional UI design.

 Technologies Used

HTML5 – For dashboard structure

CSS3 – For layout, grid/flexbox, colors, and card design

 Files Included

cards_table.html – Dashboard layout

styledashboard.css – Styling for cards, table, and background

 Features

Navigation bar for page switching

Dashboard overview section

Square cards with icons and headings

Table for displaying user records

Soft and professional color theme

Consistent alignment and spacing

 How to Run

Open cards_table.html in a browser

Dashboard loads with cards and table

 Note

Dashboard is static (HTML & CSS only)

No dynamic data or JavaScript functionality

Designed to understand UI layout and styling concepts
