Project Title

Signup Form with Validation (HTML, CSS, JavaScript)

 Project Description

This project is a modern signup form created using HTML, CSS, and JavaScript.
The form collects user details and performs client-side validation before allowing submission.

The main goal of this project is to understand:

Form design using HTML

Styling with CSS (centered card layout)

Form validation using JavaScript

Displaying error and success messages dynamically

This project is UI-based only and does not store data in a database.

 Technologies Used

HTML5 – Structure of the signup form

CSS3 – Styling and layout (card design)

JavaScript – Form validation logic

 Project Structure
newsignup.html     → Form structure
README.md          → Project documentation
signup.css         → Styling and layout
signup.js          → Validation logic


 Form Fields

The signup form includes the following fields:

First Name

Last Name

Email

Phone Number

Password

Confirm Password

Each field has its own validation and error message.

 Validation Rules

The following validations are implemented using JavaScript:

All fields are mandatory

First Name and Last Name:

Cannot be empty

Must contain only alphabets

Email must be in a valid email format

Phone number must be exactly 10 digits

Password must be at least 6 characters

Password and Confirm Password must match

Error messages are displayed below each field

Form submission is stopped if validation fails

A success message is shown if all validations pass

 Features

Clean and professional UI

Centered card layout

Individual error messages for each field

Success message on valid submission

Beginner-friendly and easy to understand code

 How to Run the Project

Download or clone the project files

Make sure all files are in the same folder

Open signup.html in any web browser

Fill in the form and test validations

No server or backend setup is required.

 Note

This project is created for learning and practice purposes

Data is not saved anywhere

JavaScript validation works only on the client side
