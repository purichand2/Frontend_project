INTERNSHIP2025 - FIRST HTML WEBPAGE

	*This project is a simple introductory webpage created using basic HTML. 
	*It is part of my learning journey as I explore web development and improve my coding skills.

Project Overview
-----------------------------
The webpage displays:

	1)A welcoming heading introducing the author

	2)A short paragraph about learning HTML

	3)A list of hobbies

	4)Contact information

	5)It also uses inline CSS for background color and text alignment.

Features
-------------------------------
	1)Basic HTML Structure (DOCTYPE, <html>, <head>, <body> sections)

	2)Inline CSS styling to change background color and text alignment

	3)Lists to display hobbies

	4)Contact details section

How to Run
