Login & Signup Page (HTML & CSS)
 Project Title

Login and Signup Page using HTML & CSS

 Project Description

This project focuses on creating a Login and Signup user interface using only HTML and CSS.
The main aim is to understand page layout, form structure, navigation, and styling without using any backend or JavaScript.

Both pages are designed to look clean, professional, and user-friendly, similar to real-world websites.

 Technologies Used

HTML5 – For page structure and form elements

CSS3 – For styling, layout, colors, and alignment

 Pages Included

login.html – User login page

signup.html – User signup/registration page

styleloginpage.css – Common styling for forms and layout

 Features

Navigation bar with links to other pages

Input fields for email, password, and user details

Clean card-style layout

Responsive and centered design

Proper labels and placeholders for inputs

 How to Run

Open the project folder

Double-click login.html or signup.html

The page will open in your web browser

 Note

This project is UI-based only

No form validation or data storage is implemented

Created for learning HTML structure and CSS styling
