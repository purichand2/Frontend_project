Registration Form (Multi-Step UI) – HTML & CSS
 Project Title

Multi-Step Registration Form using HTML & CSS

 Project Description

This project is a multi-step registration form UI created using HTML and CSS only.
The form is divided into multiple sections to collect user information in an organized manner.

The purpose of this task is to learn how to:

Break a large form into steps

Use CSS for page transitions and alignment

Design a professional-looking form layout

 Technologies Used

HTML5 – Form structure and inputs

CSS3 – Styling, layout, step indicators, and transitions

 Files Included

reg_form.html – Multi-step registration form

styleregform.css – Styling for form, steps, and layout

 Form Sections

Personal Details

Address Details

Education Details

Course Details

Preview Page

Each step is visually separated and easy to understand.

 How to Run

Open reg_form.html in a browser

Navigate through form steps using buttons

 Note

This project is UI-only (HTML & CSS)

No data validation or submission logic

Created to practice multi-step form design
